Installation Guide

av CLI is available for macOS and Linux. For GitHub integration, install both av and GitHub CLI.

macOS (Homebrew)

brew install gh aviator-co/tap/av

Arch Linux (AUR)

yay av-cli

Debian/Ubuntu

echo "deb [trusted=yes] https://apt.fury.io/aviator/ /" | sudo tee /etc/apt/sources.list.d/fury.list sudo apt update && sudo apt install av

RPM-based Systems

sudo yum install av

Configure repository: /etc/yum.repos.d/fury.repo

Binary Download

Download from releases page

chmod +x av && sudo mv av /usr/local/bin/

Upgrade Instructions

How to upgrade from previous versions

Upgrade Process

Upgrading av CLI is straightforward and does not require any configuration changes.

macOS (Homebrew)

brew update brew upgrade av

Debian/Ubuntu

sudo apt update sudo apt upgrade av

RPM-based Systems

sudo yum update av

Post-Upgrade Verification

After upgrading, verify the installation:

av --version

Test basic functionality:

av tree
💡 No Migration Required
This upgrade maintains full compatibility with existing repositories and configurations.
